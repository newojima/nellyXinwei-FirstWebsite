What is HTML 5.0?

HTML 5.0 is the latest version of Hyper Text Markup Language which is made in the year 2014.It was developed to solve compatibility problems of the older versions of HTML. The older HTML versions before HTML 5.0 was made require proprietary plugins and APIs.Because of this version was made, HTML 5.0 was able to provide interface to make loading elements easier.The design goal of this version is to support for multimedia on mobile devices. There are new syntactic features to support the version.Videos, Audio and Canvas tags are examples of the new features of this version.There are now ways users interact with the documents which includes the new parsing rules for enhanced flexibility , new attributes , elimination of outmoded or the redundant attributes.Drag and drop capabilities from document to another , Offline editing , messaging enhancements , detailed rules for parsing,MIME and protocol handler registration.

Tim Berners Lee is the one who invented the HTML ( Hyper Text Markup Language).The Latest Hyper Text Markup Language version was created by W3C or the World Wide Web Consortium with the help of the group called the WHATWG (Web Hypertext Application Technology Working Group) which is this version of HTML, HTML 5.0.Tim Berners Lee invented HTML to make a publishing language for the computer in making websites and too make it easier for people to use this language and making it simpler for them to understand. The SGML was used before HTML, before Tim Berners Lee invented it and spread his idea of what he created which was before, only for himself when he was finding a way to make a system in a way to link documents together by using buttons on the computer screen.
